# Telecom Churn Analysis

## Project Overview
This project analyzes customer churn in the telecom sector. The goal is to understand patterns and factors contributing to 
customer churn and provide actionable insights to reduce churn rates.

The analysis uses a dataset containing customer information, usage patterns, and churn history.

---

## Files in the Repository

- `Telecom_Churn_Analysis.ipynb` – Jupyter Notebook containing all data analysis, visualizations, and insights.  
- `cleaned_telecom_data.csv` – Preprocessed and cleaned dataset used for analysis.  

---

## Key Analysis Steps

1. **Data Cleaning** – Handling missing values, correcting data types, and removing duplicates.  
2. **Exploratory Data Analysis (EDA)** – Visualizing key metrics and patterns in the data.  
3. **Customer Segmentation** – Identifying groups of customers prone to churn.  
4. **Churn Prediction Insights** – Using descriptive analytics to highlight factors influencing churn.  
5. **Visualization** – Charts, graphs, and plots to communicate findings effectively.
